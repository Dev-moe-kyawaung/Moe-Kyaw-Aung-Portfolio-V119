## 🐛 Troubleshooting

**Q: Dark mode not working?**
A: Check browser's localStorage support. Clear cache and refresh.

**Q: Particles not animating?**
A: Ensure browser supports Canvas API. Update to latest browser version.

**Q: Mobile menu not showing?**
A: Check viewport meta tag is present in `<head>`.

**Q: Colors not changing?**
A: Clear browser cache or use Ctrl+Shift+Delete to clear CSS cache.

**Q: Form not submitting?**
A: Form is handled with client-side validation. Add server integration as needed.
